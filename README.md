## Como rodar?
O backend roda utilizando Go 1.17.
Primeiramente é necessária instalação das dependencias na pasta do projeto.
```
go install
```
Após isso basta utilizar seguinte comando para rodar o backend:
```
go run main.go
```

### Deploy 
```
go deploy
```
